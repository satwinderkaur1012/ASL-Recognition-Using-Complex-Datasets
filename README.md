# American-Sign-Language-Recognition-Using-Complex-Datasets
Understanding sign language is a tedious task for people like us, and communication gaps still exist between the deaf and hearing society. The only solution is automatic sign language recognition (SLR) systems that aim to translate visual hand gestures into linguistic representations. However, it is challenging to perform well in real-time environments due to noisy backgrounds, lighting effects, and imbalanced training data. The motivation of this research is to address these challenges, with the help of Vision Transformer (ViT) architectures. We focus on static American Sign Language (ASL) benchmark datasets having different levels of complex backgrounds and human-induced noise. In this proposed work, stratified K-fold cross-validation and early stopping are used to reduce overfitting and tackle class imbalance, thus promoting model robustness. From experimental analysis, the ViT-based approach shows strong validation performance but notable generalization gaps when evaluated on inference datasets with restricted variety or gesture classes that are similar in appearance. Although this is the limitation for deployment in real-time, findings reveal the importance of designing human-noise datasets that are deployment-oriented. This work describes the strengths and limitations of attention-based models and provides insights for designing a more robust framework for assistive communication applications.
